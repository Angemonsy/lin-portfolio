大学生成果证据仓

把奖学金、综测、保研、实习和简历里反复要用的成果证据，收进一个本地可运行的轻量账本。录入成果名称、适用场景、证明完整度和截止节点后，页面会自动整理缺件提醒、可提交清单和 JSON 备份，适合学期末、报名季和秋招前反复整理。

需求依据

很多大学生不是没成果，而是成果散在聊天记录、邮箱、网盘和相册里，真正填表时才发现证据链断掉了。

核心功能

成果录入、场景筛选、完整度评分、缺件提醒、申报包整理、本地草稿和 JSON 导入导出。

数据方式

纯前端静态可跑，使用 localStorage 做轻量存储，并支持 JSON 备份与恢复，后续可平滑迁移到 SQLite。
